border = borda

margin = margem
outline = contorno

tipos de caixa = box - inline
em box: toda caixa de nivel box, sempre vai pular a linha para continuar o conteudo
em inline: não pula para a proxima linha e não quebra a linha
uso da tag 'display' para transformar uma tag 'box' para inline e vise versa, box para inline 'display: inline' e inline para box 'display: block'
tendo tambem a possibilidade de 'inline-block' para que e linha tenha algumas caracteristicas dos dois elementos
